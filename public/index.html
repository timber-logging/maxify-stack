Maxify Stack by Timber Logging

Timber Logging provides simple application logging and tree puns. This project was created as a way to convert minified stack traces using source maps, it is free for anyone to use.

To use it, you will need to create the source maps from the identical code as what generated the minified stack trace.

You must add your folder paths to the folder-paths.json file. If you only have one project then you should only need one folder path.

To learn how to generate source maps, and find out more about this project, click here.
